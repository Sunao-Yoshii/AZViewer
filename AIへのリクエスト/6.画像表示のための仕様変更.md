
# 変更仕様書

## 画像一覧表示機能および検索機能追加

---

## 1. 目的

本仕様は、登録済み画像情報を一覧表示する機能および検索機能を追加し、ユーザーが効率的に画像情報を参照・管理できるようにすることを目的とする。

---

## 2. 変更概要

以下の機能を新規追加・変更する。

- `Home.vue` を画像一覧表示画面へ変更
- `SamplePage.vue` の削除
- サイドバーの役割変更（登録 + 検索UI）
- `image_file_data` を対象とした検索機能追加
- タイル形式による画像一覧表示
- ページング機能追加
- フラグ更新機能追加
- サムネイル取得API追加

---

## 5. API仕様

---

### 5.1 search_image_files

#### 概要

画像一覧検索およびページング取得を行う。

#### 入力

```json
{
  "path": "",
  "rating": null,
  "is_checked": null,
  "is_favorite": null,
  "page": 1,
  "page_size": 25,
  "sort": "id_desc"
}
````

#### 仕様

* 検索・ページングは DB 側で実行
* フロントは条件のみ送信

---

### 5.2 戻り値DTO

#### ImageFileListItem

```python
class ImageFileListItem:
    id: int
    filename: str
    path: str
    folder: str
    rating: str
    is_checked: int
    is_favorite: int
    comment: str | None
```

---

#### SearchImageFilesResult

```python
class SearchImageFilesResult:
    items: list[ImageFileListItem]
    total_count: int
    total_pages: int
    page: int
    page_size: int
```

---

### 5.3 update_image_file_flags

#### 概要

フラグ更新を行う

#### 入力

```json
{
  "id": 1,
  "field": "is_checked",
  "value": 1
}
```

#### 仕様

* 1APIで1レコード1項目更新
* 楽観更新を採用
* 成功時は再取得しない
* 検索条件に一致しなくなった場合は一覧から除去
* 失敗時は再検索実行 + トースト通知

---

### 5.4 fetchLocalImageThumb

#### 概要

サムネイル画像取得

#### 入力

```json
{
  "id": 1
}
```

#### 出力

* バイナリデータ

#### Content-Type

* 拡張子に応じて決定

  * image/png
  * image/jpeg
  * image/webp
  * image/avif

#### 縮小仕様

* 150x150 矩形内に収まるよう縦横比維持で縮小
* 拡大は行わない

#### エラー時

* 404 を返却
* フロント側で placeholder.png を表示

#### キャッシュ

* キャッシュは考慮しない（毎回取得）

---

## 7. 検索仕様

---

### 7.1 条件

| 項目          | 条件              |
| ----------- | --------------- |
| path        | 部分一致            |
| rating      | 完全一致 / 未指定で全件   |
| is_checked  | チェック時のみ true 条件 |
| is_favorite | チェック時のみ true 条件 |

---

### 7.2 挙動

* Enterキー検索なし
* 検索ボタン押下で実行
* 検索時はページ1へリセット
* 条件はページ移動時も保持

---

### 7.3 検索中

* 全画面オーバーレイ表示
* 文言：「検索中」
* 多重検索禁止
* ページ操作・ソート変更禁止

---

## 8. ソート仕様

---

### 8.1 選択肢

| 表示名      | 値             |
| -------- | ------------- |
| 登録の新しい順  | id_desc       |
| 登録の古い順   | id_asc        |
| ファイル名昇順  | filename_asc  |
| ファイル名降順  | filename_desc |
| レーティング昇順 | rating_asc    |
| レーティング降順 | rating_desc   |

---

### 8.2 初期値

```
id_desc
```

## 9. ページング仕様

---

### 9.1 ページサイズ

* 25 / 50 / 75 / 100

### 表示位置

* 一覧上部

---

### 9.2 ページリンク

* 上部・下部両方に表示

---

### 9.3 表示ルール

* 1ページのみ：非表示
* 5未満：全表示
* 5以上：前後2ページ表示

---

### 9.4 動作

* 再検索で1ページへ
* サイズ変更で1ページへ
* 0件時は非表示
